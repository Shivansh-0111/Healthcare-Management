# 🏥 Healthcare Management System

A web-based Healthcare Management System built with Django that allows 
patients and doctors to manage appointments, medical records, and 
health data efficiently.

## 🚀 Features
- Patient registration and login
- Doctor/Admin dashboard
- Appointment booking system
- Medical records management

## 🛠️ Tech Stack
- **Backend:** Python, Django
- **Database:** SQLite
- **Frontend:** HTML, CSS
